📼 RetroVerse

Compra · Venta · Intercambio retro

RetroVerse es una tienda digital dedicada a objetos vintage, coleccionables y tecnología retro. Un lugar para quienes coleccionan consolas de los 80, walkmans, cámaras analógicas y todo lo que hace sonar la nostalgia.

¿Qué es RetroVerse?

Una tienda pensada para coleccionistas y nostálgicos: comprá piezas únicas, vendé lo que ya no usás, e intercambiá tesoros retro con otros fans. Cada producto tiene su año de fabricación, estado de conservación y una historia propia.

Usá el menú de arriba para explorar el Catálogo o aprender cómo se organiza la información de cada producto en Sobre el registro.

¿Qué es un registro?

Un registro es como una "ficha" de información sobre una sola cosa. Pensalo como una tarjeta de un fichero: cada producto de RetroVerse tiene su propia tarjeta con datos como su nombre, categoría, año, precio, etc. Todas las tarjetas juntas forman el catálogo completo.

¿Qué es una clave principal?

Una clave principal (o clave primaria) es el dato que identifica a cada registro de forma única, sin que se repita nunca. En RetroVerse, esa clave es idProducto: ningún otro producto puede tener el mismo número de identificación. Así, aunque dos productos se llamen igual o cuesten lo mismo, el sistema siempre sabe exactamente de cuál se trata gracias a su idProducto.

Prompts utilizados para construir RetroVerse

Prompt 1

"Diseña un e-commerce creativo llamado RetroVerse dedicado a objetos vintage y tecnología retro. Define un registro principal Producto con campos, tipos de datos y una clave principal."

Este prompt definió la base conceptual del proyecto: el nombre, la temática y, sobre todo, la estructura de datos. A partir de él se decidieron los ocho campos del registro Producto y se eligió idProducto como clave principal.

Prompt 2

"Genera una aplicación web HTML, CSS y JavaScript para RetroVerse con catálogo, búsqueda y detalle de productos."

Este prompt pidió convertir el diseño conceptual en una aplicación funcional. Con él se generó el código de esta página: la grilla de productos, el buscador en tiempo real y la vista de detalle de cada artículo, todo con estilo visual retro de los años 80 y 90.

Prompt 3

"Explica qué es un registro y una clave principal para estudiantes de primer año de programación."

Este prompt buscó que los conceptos técnicos fueran accesibles para quienes recién empiezan a programar. Por eso, en esta sección se explican el registro como una "ficha" y la clave principal como el dato que nunca se repite, usando ejemplos concretos del propio catálogo en lugar de definiciones abstractas.

Reflexión: ¿cómo ayudó la IA en este proyecto?

La inteligencia artificial colaboró diseñando la estructura de datos (definiendo qué campos eran necesarios y por qué idProducto debía ser la clave principal), generando el código de la aplicación web completa, redactando las explicaciones en lenguaje sencillo y creando los datos de ejemplo. Esto permitió pasar de una idea general a una aplicación funcional en mucho menos tiempo, dejando a la persona el rol de definir el concepto, revisar el resultado y tomar las decisiones finales sobre el diseño y el contenido.
